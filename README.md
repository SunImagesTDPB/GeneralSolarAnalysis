# GeneralSolarAnalysis

## Введение

Представленный инструмент - это интерактивный виджет, реализованный на Python, позволяющий строить графики зависимости интенсивности от времени вдоль корональных петель на Солнце. Построение сплайнов производится на самих солнечных AIA-изображениях, которые могут быть загружены из готового датасета. 

Данное ПО реализовано на: 
* **Linux (Ubuntu 22.04)**
* **Python 3.13.9 (с необходимыми библиотеками)**
* **VS code**
